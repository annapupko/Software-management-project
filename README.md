## General info

This project is a website for managing a queue system. 

## Technologies
Project is created with:

* Node.js version: 14.17.6 (and up)

## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm install express
$ npm install express-session
$ npm install nodemon
$ npm run dev
$ npm run angular
```
